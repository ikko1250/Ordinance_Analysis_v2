reiki-html-fetcher
reiki-searcher-playwright
2つのagentを使用して、ordinance_names.csvに記載されている条例の施行規則HTMLファイルを取得せよ。
まずは、reiki-searcher-playwrightが施行規則のURLを取得する。そのURLを使用して、reiki-html-fetcherがHTMLを取得する。
5つづつの別々のサブエージェントを作成して、並列で処理せよ。


python run_agent.py \
  --goal "「伊豆市自然環境等と再生可能エネルギー発電事業との調和に関する条例」の施行規則ページを特定し、そのHTMLを保存する" \
  --query "伊豆市自然環境等と再生可能エネルギー発電事業との調和に関する条例 施行規則" \
  --headless \
  --out-dir out

  python run_agent.py \
  --goal "「伊豆市自然環境等と再生可能エネルギー発電事業との調和に関する条例」の施行規則ページを特定し、そのHTMLを保存する。(HTML形式の施行規則を取得するためには、例規集を使用することが推奨される。各自治体の例規集については、https://uub.jp/opm/ml_reikishu.htmlに列挙されている。)" \
  --query "伊豆市自然環境等と再生可能エネルギー発電事業との調和に関する条例 施行規則" \
  --out-dir out


reiki-municipality-fetcherを使用して、ordinance_names.csvに記載されている条例の施行規則HTMLファイルを保存せよ。サブエージェントを複数起動して並列で処理せよ。ただし、起動するサブエージェントは5つに制限する。1つの条例につき1つのサブエージェントを使用すること。
